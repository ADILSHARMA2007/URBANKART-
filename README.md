## ü§î About The Project

Have you ever wanted to build your own little eBay or Amazon? UrbanKart is a project that explores the fundamentals of an e-commerce platform in a desktop environment. It's a great example of how to use Java Swing for more than just simple tools. Users can log in, manage products, and engage in buying or selling.

The goal was to create a simple yet functional application that demonstrates key software engineering principles like object-oriented design and UI development.

---

## ‚ú® Features

UrbanKart comes with a set of cool features:

*   **User Authentication:** Secure login system for users.
*   **Product Listings:** Users can view a list of available products.
*   **Sell Your Stuff:** Easily put your own items up for sale.
*   **Two Ways to Sell:**
    *   **Direct Sale:** Set a fixed price for your items.
    *   **Auction-Style Bidding:** Set a starting price and let users bid on your product!
*   **Product Management:** Edit the details of your products after listing them.
*   **Native Look & Feel:** The app adapts to the look and feel of your operating system (Windows, macOS, or Linux).

## üöÄ Getting Started

To get a local copy up and running, just follow these simple steps.

### Prerequisites

Make sure you have the following installed on your machine:

*   **JDK (Java Development Kit)** - This project uses JDK 23, but it should be compatible with other recent versions.
*   **Apache Maven** - To build the project and manage dependencies.

### Installation & Running

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/ADILSHARMA2007/URBANKART.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd UrbanKart
    ```
3.  **Build the project with Maven:**
    ```sh
    mvn clean install
    ```
4.  **Run the application:**
    ```sh
    java -jar target/UrbanKart-1.0-SNAPSHOT.jar
    ```
    (You might need to adjust the JAR file name based on the version in your `pom.xml`)
